# tg_bot_for_reaviz

Telegram-бот для прорешивания тестов из файла `anatomy_test.xlsx`.

## Что умеет

- показывает кнопку `Начать тест`
- предлагает выбрать количество вопросов
- берёт случайные вопросы из Excel-файла
- показывает варианты ответов кнопками
- сразу сообщает, верно или неверно
- при ошибке показывает правильный ответ текстом, а не только буквой
- поддерживает вопросы с несколькими правильными ответами
- поддерживает задания на соответствие и последовательность
- умеет останавливать тест

## Запуск

1. Установить зависимости:

```powershell
python -m pip install -r requirements.txt
```

2. Задать токен бота:

```powershell
$env:TELEGRAM_BOT_TOKEN="ВАШ_ТОКЕН"
```

3. Запустить:

```powershell
python bot.py
```

## Формат Excel

- 1 столбец: вопрос
- следующие столбцы: варианты ответов
- последний столбец: правильный ответ

### Логика правильного ответа

- одна буква: один правильный ответ
- несколько букв: несколько правильных ответов
- буквы с цифрами или только цифры: задание на соответствие или последовательность
